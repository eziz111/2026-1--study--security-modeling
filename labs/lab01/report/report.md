# Теоретические сведения

Экспоненциальный рост — это процесс увеличения некоторой величины, при котором скорость роста пропорциональна текущему значению этой величины.

Математическая модель описывается дифференциальным уравнением:

du/dt = αu

где:

u(t) — значение величины в момент времени t
α — коэффициент роста

Если α > 0, система демонстрирует рост.
Если α < 0, происходит экспоненциальное затухание.

Аналитическое решение данного уравнения имеет вид:

u(t) = u₀ e^(αt)

где:

u₀ — начальное значение.

# Программная реализация

Дифференциальное уравнение реализуется следующим образом:

```julia
function exponential_growth!(du, u, p, t)
    α = p
    du[1] = α * u[1]
end
```

Начальные параметры модели:

```julia
u0 = [1.0]
α = 0.3
tspan = (0.0, 10.0)
```

Решение задачи выполняется при помощи функции:

```julia
prob = ODEProblem(exponential_growth!, u0, tspan, α)
sol = solve(prob)
```
