# Цель работы

Изучить модель экспоненциального роста и реализовать её численное решение на языке Julia.

---

# Математическая модель

Дифференциальное уравнение:

du/dt = αu

где

u — значение величины
t — время
α — коэффициент роста

---

# Аналитическое решение

Решение уравнения имеет вид:

u(t) = u₀ e^(αt)

Если α > 0 — происходит экспоненциальный рост.

---

# Реализация в Julia

Используемые библиотеки:

- DifferentialEquations
- Plots
- DataFrames
- DrWatson

---

# Основной код

```julia
function exponential_growth!(du, u, p, t)
    α = p
    du[1] = α * u[1]
end
```
