Модель экспоненциального роста  
Бердыев Эзиз НФИмд-01-25  
Цель работы  
Изучить модель экспоненциального роста и реализовать её численное решение на языке Julia.  
Математическая модель  
Дифференциальное уравнение:  
du/dt = αu  
где  
u — значение величины t — время α — коэффициент роста  
Аналитическое решение  
Решение уравнения имеет вид:  
u(t) = u₀ e^(αt)  
Если α > 0 — происходит экспоненциальный рост.  
Реализация в Julia  
Используемые библиотеки:  
• DifferentialEquations  
• Plots  
• DataFrames  
• DrWatson  
Основной код  
function exponential_growth!(du, u, p, t)  
α = p  
du[1] = α * u[1]  
end  
Параметры модели  
Начальные параметры:  
u0 = 1 α = 0.3 t = 0..10  
Результаты моделирования  
В результате решения был получен график экспоненциального роста.  
Время удвоения  
Формула:  
T = ln(2) / α  
Для α = 0.3  
T ≈ 2.31
